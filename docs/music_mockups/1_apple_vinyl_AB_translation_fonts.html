🎯 我的最终推荐

  • 翻译：模式三（每行原文 + 当前句下挂一条小字译文），设置里给「翻译开/关」+「显示原文/译文/双语」三选项。
  • 字体：F3 Inter + Noto Sans SC 作主力，配 F1 Microsoft YaHei UI 作系统回退；用户可在设置里切到 F6 (HarmonyOS) 或 F7 (霞鹜文楷)。
  • 实现：在 resources/fonts/ 放入打包字体，用 IDWriteFontCollection 自定义集合加载，避免依赖系统安装。
